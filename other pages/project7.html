PASSCODE


This is an hypothetical iOS application that I created using Adobe Experience Design. With the rising increase in the number of invasion of privacy incidents, I set out to create a configuration of an app that could be used to store important data, such as credit card information, internet logins, and passwords. After creating the design and assembling a prototype in Adobe XD, I later edited the final images using Adobe Photoshop.
